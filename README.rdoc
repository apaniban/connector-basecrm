=== BaseCRM Connector

== README

Application connector between Connec!™ and BaseCRM.

== Summary

Connec!™ is Maestrano's patented data sharing system. It is a hub and spoke API hub that keeps in sync applications connected to it. Thanks to Connec!™ data entered into one system (e.g.: accounting package) are automatically propagated to your other systems (e.g.: customer management application).
